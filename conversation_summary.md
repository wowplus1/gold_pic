# 주얼리 해리율(Gold Loss) 대시보드 구축 요약

## 1. 개요
사용자의 요청에 따라 주얼리 제조 과정(주물, 세공, 최종 광택)에서 발생하는 '금 손실(해리율)'을 추적하고 자동 계산하는 모던한 웹 대시보드를 구축했습니다.

## 2. 주요 기능 및 구현 내용
- **Top Section (상단 KPI 카드)**: 총 디자인 수, 총 초기 중량, 평균 해리율, 총 손실량을 실시간으로 업데이트하여 보여줍니다.
- **Middle Section (스마트 입력 폼)**: 디자인 이름과 3단계 공정별 중량을 입력하는 폼입니다. 중량을 입력할 때마다 실시간으로 예상 손실량과 해리율을 하단에 자동 계산하여 미리 보여줍니다.
- **Bottom Section (데이터 원장 및 분석 테이블)**: 입력된 모든 디자인의 공정 기록과 손실 데이터를 테이블 형태로 제공합니다. 5% 이상의 높은 해리율이 발생하면 빨간색 하이라이트와 경고 아이콘으로 표시합니다. CSV 다운로드 기능도 포함되어 있습니다.

## 3. 디자인 시스템
- React/Tailwind CSS 스타일을 차용하여 Vanilla HTML/CSS/JS 로 빠르고 가벼운 앱을 구성했습니다.
- **색상**: 깔끔한 흰색(White)과 밝은 회색(Light Gray) 배경에, 강조색으로 깊은 네이비 블루(Deep Navy Blue)와 골드/앰버(Amber)를 사용했습니다.
- **타이포그래피**: 테이블 숫자가 깔끔하게 정렬되도록 `tabular-nums` 속성을 적용한 Sans-serif(Inter 폰트) 폰트를 도입했습니다.

## 4. 파일 구성
현재 폴더(`c:/Users/조지수/.gemini/헤리율계산`)에 다음 파일들이 생성되었습니다.
1. `index.html`: 대시보드 마크업 구조
2. `style.css`: UI/UX 디자인 및 반응형 스타일
3. `app.js`: 데이터 처리, 실시간 자동 계산 로직 및 동적 UI 렌더링 스크립트
